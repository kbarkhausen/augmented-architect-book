## Project: FlowTrack

A modern issue tracking system built as a learning project for AI-augmented development. Features user management, issue lifecycle, and GitHub integration.

## Tech Stack
- Node.js 20, TypeScript 5.4
- Express 4.18 (API)
- React 18, Vite 5 (Frontend)
- Prisma 5.x (PostgreSQL)
- Vitest + React Testing Library
- Deployed on Railway
