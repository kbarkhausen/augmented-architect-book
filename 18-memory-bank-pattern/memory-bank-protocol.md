# CLAUDE.md

## Memory Bank Protocol

### At Session Start
1. Read `memory-bank/activeContext.md` for current work
2. Read `memory-bank/progress.md` for status
3. Check `memory-bank/decisionLog.md` before architectural choices

### When Finishing Work
1. Update `activeContext.md` with new status
2. Check off completed items in `progress.md`
3. Add significant decisions to `decisionLog.md`
