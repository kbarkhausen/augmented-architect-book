# AGENTS.md

## Project: TechFlow CRM

A customer relationship management system for B2B SaaS companies.

## Tech Stack

- **Frontend**: React 18, TypeScript 5.4, Vite, TailwindCSS
- **Backend**: Node.js 20, Express 4, TypeScript
- **Database**: PostgreSQL 15 via Prisma
- **Cache**: Redis 7
- **Queue**: BullMQ
- **Testing**: Vitest, Playwright
- **Deployment**: Docker, AWS ECS
